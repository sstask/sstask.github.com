mysql主服务器的安装

1.yum -y install mysql
2.yum -y install mysql-server
3.yum -y install mysql-devel
4.按需求修改mysql配置/etc/my.cnf

如修改数据库文件存放位置datadir = $mysql_data_dir
修改日志文件位置log_bin = mysql_log_dir/mysql-bin.log log-error = mysql_log_dir/mysqld.log

5.自启动chkconfig mysqld on
6.设置root密码 /usr/bin/mysqladmin -u root password “root_pwd”
7.添加用户GRANT all privileges on . to ‘username’@’%’ identified by ‘pwd’ WITH GRANT OPTION;flush privileges;

mysql从服务器安装

1.yum -y install mysql
2.yum -y install mysql-server
3.yum -y install mysql-devel
4.修改mysql配置/etc/my.cnf

5.启动
/usr/bin/mysql_install_db –datadir=/mysql/3307/data –user=mysql
/usr/bin/mysqld_multi start 3307
6.设置root密码 /usr/bin/mysqladmin -uroot password “root_pwd” -S /mysql/3307/data/mysql.sock
7.在主mysql服务器中添加slave账号，记录bin日志位置
8.change master to master_host=……
